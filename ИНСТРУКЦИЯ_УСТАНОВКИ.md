# Инструкция по сборке и установке Secure Vault

## Способ 1: GitHub + GitHub Actions (БЕЗ КОМПЬЮТЕРА)

Это наиболее простой способ получить APK без компьютера.

### Шаг 1 — Загрузите код на GitHub
1. Зарегистрируйтесь на https://github.com (бесплатно)
2. Создайте новый репозиторий: нажмите «+» → «New repository»
3. Назовите его `SecureVault`, выберите Private (приватный)
4. Нажмите «Create repository»

### Шаг 2 — Загрузите файлы проекта
1. На странице репозитория нажмите «uploading an existing file»
2. Перетащите всё содержимое папки `SecureVault` (все файлы и папки)
3. Нажмите «Commit changes»

### Шаг 3 — Создайте workflow для сборки
1. В репозитории нажмите «Add file» → «Create new file»
2. Назовите файл: `.github/workflows/build.yml`
3. Вставьте следующий код:

```yaml
name: Build APK

on:
  push:
    branches: [ main ]
  workflow_dispatch:

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      
      - name: Set up JDK 17
        uses: actions/setup-java@v4
        with:
          java-version: '17'
          distribution: 'temurin'
      
      - name: Setup Android SDK
        uses: android-actions/setup-android@v3
      
      - name: Make gradlew executable
        run: chmod +x gradlew
      
      - name: Build Debug APK
        run: ./gradlew assembleDebug
      
      - name: Upload APK
        uses: actions/upload-artifact@v4
        with:
          name: SecureVault-debug
          path: app/build/outputs/apk/debug/app-debug.apk
          retention-days: 30
```

4. Нажмите «Commit new file»

### Шаг 4 — Скачайте APK
1. В репозитории нажмите вкладку «Actions»
2. Нажмите на запущенный workflow «Build APK»
3. Дождитесь завершения (3-5 минут, зелёная галочка)
4. В разделе «Artifacts» нажмите «SecureVault-debug»
5. Скачается файл `SecureVault-debug.zip` с APK внутри

### Шаг 5 — Установите APK на телефон
1. На телефоне Android: Настройки → Безопасность → «Установка из неизвестных источников» → включить
   (или: Настройки → Приложения → разрешить установку из браузера/файлового менеджера)
2. Откройте скачанный ZIP → распакуйте → откройте `app-debug.apk`
3. Нажмите «Установить»

## Способ 2: Replit (онлайн-IDE, с телефона)

1. Зайдите на https://replit.com с телефона
2. Создайте аккаунт
3. Нажмите «Create Repl» → выберите «Android (Kotlin)» (если доступно)
4. Загрузите файлы проекта
5. Запустите сборку — Replit выдаст APK

---

## Способ 3: Buddybuild / Appcircle (облачная CI/CD)

Если GitHub Actions по каким-то причинам не подходит:

1. Зайдите на https://appcircle.io (есть бесплатный тариф)
2. Подключите GitHub-репозиторий
3. Выберите проект Android
4. Нажмите «Build» — получите APK ссылкой

---

## Разрешение на установку из неизвестных источников

### Android 8.0 и выше (большинство телефонов):
- Настройки → Приложения → специальный доступ → Установка неизвестных приложений
- Выберите браузер или файловый менеджер → включите

### Samsung:
- Настройки → Биометрия и безопасность → Установка неизвестных приложений

### Xiaomi (MIUI):
- Настройки → Конфиденциальность → Специальные разрешения → Установка из неизвестных источников

### Старые Android (до 8.0):
- Настройки → Безопасность → Неизвестные источники → включить

## Системные требования

- Android 8.0 (API 26) и выше
- ~15 МБ свободного места
- Для биометрии: телефон с отпечатком пальца

## Важно о безопасности

- Запомните мастер-пароль — восстановить его НЕВОЗМОЖНО
- При потере мастер-пароля все данные будут недоступны (это задумано — защита от взлома)
- Не делайте резервные копии файла базы данных без защиты
- APK подписан debug-ключом — для production нужно настроить keystore в secrets GitHub
